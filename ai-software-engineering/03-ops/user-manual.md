# AnyDownloader（Web）— 用户说明

## 启动

在 `any-downloader-vue` 目录执行 `npm install` 后：

- 开发预览：`npm run dev`，按终端提示在浏览器打开本地地址。
- 构建静态站：`npm run build`，产物在 `dist/`，可由任意静态服务器托管。

## 基本操作

1. 在**工具栏**输入或粘贴资源地址，点击 **添加任务**（或菜单「文件 → 添加地址」）。
2. 在列表中**选中**一行，点击 **开始**；**暂停**会中断当前 HTTP 拉取。
3. 右侧 **Dock**：**下载信息**、**网络测速**、**下载块图**；右缘竖条按钮可切换各面板显示；**窗口**菜单亦可切换。
4. **状态栏**显示就绪与任务数摘要；**点击状态栏**打开 **日志**，可 **复制全文**。
5. **F1** 打开帮助；**帮助 → 关于** 查看版本说明。

## 语言与主题

- **语言**菜单：**中文** / **English**。
- **主题**菜单：**跟随系统**、**浅色**、**深色**（会记住到本机浏览器存储）。

## 协议说明

- **HTTP/HTTPS** 与 **data:** 可在浏览器内直接尝试下载（受目标站点 **CORS** 与混合内容策略限制）。
- **FTP、Magnet、eD2k** 等会识别协议并列入任务，但开始下载时会提示需 **桌面壳或后端引擎**（当前为占位）。

## 常见问题

- **一直失败 / 提示网络或 CORS**：多为跨域资源不允许浏览器脚本读取，需同域资源、目标允许 CORS，或使用桌面下载器/代理方案。
